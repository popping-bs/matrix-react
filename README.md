# Semiconductor Dashboard

반도체 공정 데이터 분석용 내부 대시보드 UI 프로토타입 프로젝트입니다.

## 기술 스택

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui 스타일 컴포넌트

## 로컬 실행 방법

```bash
git clone <repo-url>
cd web
npm install
npm run dev
```

개발 서버가 실행되면 터미널에 표시되는 로컬 주소로 접속하면 됩니다.

## 사용 가능한 스크립트

```bash
npm run dev
npm run build
npm run preview
```

## 현재 구현 범위

- 대시보드 기본 레이아웃
- Navbar
- Sidebar
- Analysis 페이지 프로토타입
